NoExperienceMod is a plugin that disables the new vanilla experience in 1.3, allowing mods such 
as ExperienceMod to change the experience system correctly.

    
TO INSTALL

    Copy the NoExperienceMod.jar file to the Bukkit plugin folder.

    
HOW TO COMPILE

    If it's not already referenced, include CraftBukkit #2326 or later:
        craftbukkit-1.3.1-R1.1-20120809.005138-6.jar
